This repository contains the stimuli and the code used for the study "Acoustic Influences on Human Detection of Synthetic Speech" by Hannah Cohen and Aurelija Spunde for the Perception and Action exam.

CONTENTS
- stimuli: 84 files in total selected from the ASVspoof Challenge 2019 logical access (LA) dataset’s development folder (Yamagishi et al., 2019)
  - real
      - easy
      - hard
  - fake
      - easy
      - hard
  - practice_real
  - practice_fake
- code for the PsychoPy experiment: deepfake_recognition_experiment.py
- code for the data analysis:
  - Markdown file: data_analysis.Rmd
  - Knitted PDF version: data_analysis.pdf
